How to edit template

  • - Change intro background image.
    Upload your image to images/bg/ folder. Open style.css file and find .header selector.
  • In url('http://via.placeholdr.com/1920x1080') change url value to your image path. F.e. url("../images/bg/image.jpg")

  • - Change second background image.
    Upload your image to images/bg/ folder. Open style.css file and find .counter__content selector.
  • In url('https://via.placeholder.com/1920x1080') change url value to your image path. F.e. url("../images/bg/image2.jpg")

  • - Change third background image.
    Upload your image to images/bg/ folder. Open style.css file and find .feedback_content selector.
  • In url('https://via.placeholder.com/1920x1080') change url value to your image path. F.e. url("../images/bg/image2.jpg")

  • - Change Video background.
    Upload your image to images/bg/ folder. Open index.html file and find .fillWidth selector.
  • In src="http://placehold.it/1920x1080" change url value to your video path. F.e. src="images/bg/video_bg.mp4"

  • - Change other images.
    Upload your image to images/ folder. Open index.html file and find img tag. Change the url to your image path.
  • - All texts you can change in index.html files
  • - How to edit contact form.
    Open file index.html Find .contact-form selector
  • In value="yourEmail@gmail.com" change value to email which you need.

  • - How to edit Happy Clients.
    Open file index.html Find .counter__content selector
  • In data-num="your number" change value to a number which you need.

  • - How to edit My Skills.
    Open file index.html Find .mySkills selector
  • In data-num="your number" change value to a number which you need.

  • - How to edit self-written text.
    Open file typing.js Find strings
  • In ["Here is your text"] change value to a text which you need.

  • - How to edit download button.
    Upload your file to 01_Parallax folder. strings
  • Open index.html file and find .btn__aboutMe class.

  • Change the href to your file path.

  • - How to edit social link.
    Open file index.html Find .social and .link_social_aboutMe and .contact-social class.
  • In href="#" change the href value to your link path.

  • - How to edit main color.
    Open file style.css Find :root 34 line of code
  • In # change value to a color which you need.
